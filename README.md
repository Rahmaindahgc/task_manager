# UTS Task Manager API

Project ini merupakan backend sederhana untuk aplikasi Task Manager menggunakan Node.js dan PostgreSQL.

---

## Teknologi yang Digunakan
- Node.js
- Express.js
- PostgreSQL
- pg (PostgreSQL client)

---

## Fitur Utama
- Menampilkan semua task (GET /tasks)
- Menampilkan task berdasarkan ID (GET /tasks/:id)
- Menambahkan task (POST /tasks)
- Mengupdate task (PUT /tasks/:id)
- Menghapus task (DELETE /tasks/:id)
- Validasi input (status 400)
- Error handling (status 404)
- Middleware logging
